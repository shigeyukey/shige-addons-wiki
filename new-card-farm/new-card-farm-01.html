[ üìñ How To Use ]

This Add-on is a cusimized version of the New Card Heatmap (Not Free) . You can grow crops and flowers with the new cards you have learned.



You can change the crop.


Supports night mode.


You can open options by clicking on the button with the name of the crop.


[ ‚öô Options ]


[ üåª Flower ]

  1. Sunflower, Rose, Tulip, Bird of paradise, Chamomile, Hyacinth, Hydrangea, Lotus, Orchid

  2. Max 20 new cards.

[ üéÉ Vegetable ]

  1. Carrot, Potato, Pumpkin, Tomato

  2. Max 30 new cards.

[ Size : 10px - 32px ]

  1. You can change the size of the display. If set to 10px, it will be the same size as the heatmap. The default setting is 20px.

[ Calculation ]

  1. Add new cards does not count, learning new cards counts.

  2. When the new card interval becomes the next day or later, the number is counted.

  3. If the card does not graduate, it will not be counted. If it graduates the next day, it will be counted then.

[ Notes ]

  1. Show number of New cards learned today.

  2. Show longest streak. (New cards only)

  3. Show current streak. (New cards only)

  4. Mouse hover tooltip shows date and number of cards.

  5. This add-on is relatively simple code, so it does not have advanced features like the Review Heatmap. (This add-on is not a Fork or customized version)

  6. Old Anki is not supported. (like Anki2.1.35 - 2.1.45)

[ Download ]

  1. Download the .ankiaddon file and open it with a double-click, or install it from the "Install from file" menu in the Anki add-ons management screen. (Please note that you cannot install the file by dragging and dropping it into Anki) If you encounter any errors, please contact me.
