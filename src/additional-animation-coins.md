# 🎖️Additional animation coins

This Anki add-on is for Patrons only (not Free), it adds animated coins to the Anki KillStreaks add-on. Other than this animated coins it works the same as normal KillStreaks. How to use KillStreaks please read the Wiki: [(Free)🎖️Anki KillStreaks](anki-killstreaks.md)



![Anki-animation-Coins](https://shigeyukey.github.io/shige-addons-wiki/images/Anki-killstreaks/killstreaks_animated_coins.webp)

## 📖How to use

If you become a Patron and support my volunteer development you can use this add-on. Revenue generated by this add-on will be used to develop free Killstreaks.

1. After downloading the add-on from Patreon "Anki Coins" will be added to the theme.
1. Select "Anki Coins" to use it.
1. The coins will rotate on the deck screen.
1. In the review screen, the coins do not rotate, they are static.
1. These coins are my original.
