## How to use

In the reviewer, display text animation feedback based on your answers. There are features to display card type, due date, and ease (or difficulty). This add-on is an improved and enhanced version of the Ease Scouter I previously developed.

The add-on will activate automatically after installation, and text feedback will be displayed when you review.

## Options

Text feedback can be customized with options. This option can be accessed from the add-ons menu. (Anki Menu -> Tools -> Anser text feedback)

### Front Options

You can customize the text displayed on the reviewer's Front. (Question)

![alt text](https://shigeyukey.github.io/shige-addons-wiki/images/answer-text-feedback/00.png)

* Show front question text
    * Show or hide all text on the front.
* Show answer grade
    * Show or hide the answer grade text. (Again, Hard, Good, Easy)
* Show Interval (Front)
    * Show the interval of the card after answering. (e.g. 10m, 1dÔºå1mo)
* Front font size
    * Adjust the font size.
* Front fade time (ms)
    * Adjust the time until the text is hidden.
    * This value is in milliseconds. (e.g. 1000ms -> 1 second)
* Front text position
    * You can set the text position to one of nine locations, such as top, bottom, left, or right, etc., or set it randomly.
* Animation type (Front)
    * You can specify text animations such as bounce or slide-in.
    * The default is random.
    * To disable animations please use the options in the Global tab.


### Back Options

You can customize the text displayed on the reviewer's Back. (Answer)

![alt text](https://shigeyukey.github.io/shige-addons-wiki/images/answer-text-feedback/01.png)

* show back answer text
    * Show or hide all text on the back.
* show interval (back)
    * Show the number of days in the current card's interval. (e.g. 10m, 1dÔºå1mo)
* show card status balue
    * Show the card type. (e.g. New, Review, Learning)
* show current card ease (or difficulty)
    * Show the percentage for Ease (E) or Difficulty (D), which indicates the card's difficulty level.
* show Ease factor text (or difficulty)
    * Show or hide the text for E or D. (e.g. E250% or 250%)
* Back font size
    * Adjust the font size.
* Back fade time (ms)
    * Adjust the time until the text is hidden.
    * This value is in milliseconds. (e.g. 1000ms -> 1 second)
* Back text positon
    * You can set the text position to one of nine locations, such as top, bottom, left, or right, etc., or set it randomly.
* Animation type (Back)
    * You can specify text animations such as bounce or slide-in.
    * The default is random.
    * To disable animations please use the options in the Global tab.

### Global Options

These options affect both the Front and Back.

![alt text](https://shigeyukey.github.io/shige-addons-wiki/images/answer-text-feedback/02.png)

* Choose Fonts
  * The text feedback font randomly changes from over 40 fonts built-in for English. You can specify the font to use with this option. Fonts installed on your PC can be used, so if you want to use a new font, try installing the font on your PC.
  * ![alt text](https://shigeyukey.github.io/shige-addons-wiki/images/answer-text-feedback/03.png)

* Translate
    * Auto translates shown text into your Anki settings language. Disabling this will display text feedback in English.
* Bold font
    * Make the text bold. This will not work if the font does not have a bold style.
* Neon Effect (not animation)
    * Adds a neon effect to text. This feature is still a prototype and is incompatible with text animation so enabling this will auto disable animation.
* Always on top
    * Change the function to show text on top. This may help if text isn't showing properly. This option requires restarting Anki.
* Show Textures
    * Decorate text with a texture image. This texture changes randomly based on the answer. Disabling this displays text in plain color.
* Enable Animation
    * Disable or enable all text animations for Front and Back.
* Opacity
    * Set the text transparency. e.g. 0.5 is 50% semi-transparent, 1.0 is 100% fully opaque.
