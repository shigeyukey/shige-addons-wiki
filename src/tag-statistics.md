This is a fixed and customized Add-on for the latest version of Anki (Anki23 ~ Anki25+), based on the Tag Statistics Add-on which is out of development. It can display card statistics by tag. (retention, average time, average interval)

### üìñ How to use

1. Select from the tools to view Statistics. AnkiMenu -> Tools -> Tag Statistics
2. If your decks are too large, the calculation will take a while. The window will be displayed after the calculation is finished.
