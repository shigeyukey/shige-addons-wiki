This is a small add-on to disable auto update of add-ons that are displayed when you start Anki.

### üìñHow to use:

1. Download this add-on to activate it.
1. You can turn it ON/OFF by using the menu added to the Tools of the Add-on.
1. Only auto update is disabled, so you can update the add-ons manually from the add-ons dialog.
1. I tested this only with Anki25.02.
If it does not work with other versions of Anki, please contact me.
