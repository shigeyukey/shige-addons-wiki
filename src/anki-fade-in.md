# 👻Anki Fade In
👻Anki Fade In - add fade in effect to Reviewer Home etc (Custom by Shigeඞ)


**[AnkiWeb Page](https://ankiweb.net/shared/info/33018515) | Code : `33018515`**


[![banner_shigeyuki_2](https://github.com/shigeyukey/Pokemanki-Gold/assets/124401518/8408c164-e95c-4e40-98c1-393b03e04bcb)](https://www.reddit.com/user/Shige-yuki)

Adds a fade in effect when screens change in Anki to smooth out the animation, this may help as a workaround when flickering occurs on Anki screens (e.g. Zoom add-on).

This fade in feature was inspired by the add-on *"[Advanced Answer Sounds](https://ankiweb.net/shared/info/1167194350)"* so some of the code is credited to them.

## How to use

Download the add-on to enable the fade in. You can change the fade in duration in the options or disable it.

## Option


Reviewer and other fade in adjustments.

![alt text](https://shigeyukey.github.io/shige-addons-wiki/images/anki-fade-in/00.png)


### Reviewer fade in


* Reviewer Fade in
    * Disabling this will turn off all reviewer fade-ins.

* Answer Fade in
    * Disabling this will only run the fade in on Question(Front) and disable the fade in on Answer(Back).

* Reviewer fade in Span
    * Sets the duration of the reviewer fade-in in seconds. e.g. if set to 1.0 it will fade in slowly over 1 second.

* Reviewer fade in Delay
    * Sets the delay before the reviewer fade in starts in seconds. e.g. if set to 1.0 the page will remain blank for 1 second and then start the fade in.


### Others fade in

All fade in effects except for the Reviewer.


* Othres fade in
    * Disabling this will disable all fade ins except for the Reviewer.
* Othres fade in Span
    * Sets the duration of the Anki fade-in in seconds. e.g. if set to 1.0 it will fade in slowly over 1 second.
* Othres fade in Delay
    * Sets the delay before Anki fade in starts in seconds. e.g. if set to 1.0 the page will remain blank for 1 second and then start the fade in.


## Note

1. This add-on has a feature that hides the Anki screen for fade in, so if a problem occurs the Anki screen may become invisible. If so and if you cannot operate Anki please press the Shift key while launching Anki and disable the add-on.
2. If a fade in effect already exists in your card template or other add-ons it may interfere with this add-on and cause problems. If so please report the problem to me and send me the card template or add-on name.
3. If your Anki version is a bit old or unsupported the fade in will stop automatically. If so please send me your Anki version and OS name (Win, Mac, Linux).

## 📥 How do I install this add-on?

1. This add-on works with Anki for desktop (it does not work on mobile). Download Anki from the [official website](https://apps.ankiweb.net/) if you haven't already.
2. Copy and paste the add-on code ( `33018515` )  into Anki and you can install it. ( *Menu -> Tools -> Add-ons -> Get Add-ons -> Code \[ add-on code ]* )
3. When I develop bug fixes, create new features, or compatibility for New Anki, I will notify you and you can install it.
4. Add-ons will be broken when the official Anki gets a major update, so if you like this add-on please support my volunteer development by rating, sharing, and donating. Thank you!
