This add-on is a fixed and customize version of Zoom 2.1 for Anki24+. (Fork)<br>
  
 **💡Inspired add-ons :**
  1. add-on : [Zoom 2.1](https://ankiweb.net/shared/info/1846592880) / Author : [Krassowski](https://github.com/krassowski)<br>
  2. add-on : [Anki Zoom](https://ankiweb.net/shared/info/538879081) / Author : [ccz-2](https://github.com/ccz-2)<br>

**📖How to use :**  

1. Zoom values for Reviewer, Home, Overview, Editor, Stats can be memorized.<br>
This effect is valid after restarting Anki.<br>
1. Auto saving of zoom levels with Ctrl + Scroll Wheel. (Reviewer, Home, Overview, Editor, Stats.)
1. Separate zoom levels for Question and Answer, it can be disabled in options.
1. Use shortcut keys to zoom in, zoom out, Reset.(Reviewer, Home, Overview.)<br>
1. Added option to manually set zoom.(Disable auto save.)<br>


**⚙Options :**

![image](https://raw.githubusercontent.com/shigeyukey/my_addons/main/media_files/zoom%2024.gif)<br>

    1. Option window
        (Tools -> Zoom for Anki 24)
    2. custom options for shortcut keys

<br>
Default Shortcut Keys (Home and Reviewer)<br>

 1. Zoom_in : `Ctrl+Shift++`
 2. Zoom_out : `Ctrl+Shift+-`
 3. Reset : `Ctrl+Shift+^`

**🚨Known issue:**  

1. Incremental reading add-on is incompatible. (Incremental read add-on has its own zoom).
1. "`Ctrl++`" or "`Ctrl+-`" will not work, please set another Shortcut Keys.

**Others**
1. I added simple support for old Anki (Anki2.1.35-Anki2.1.44), but some new features are not available. 
1. I added a workaround for a crash problem that was occurring with another zoom addons, so it is relatively safe.
1. If the flicker problem occurs the Fade In add-on might help alleviate it a bit. addon: <a href="https://ankiweb.net/shared/info/33018515" target="_blank">👻Anki Fade In (by Shigeඞ)</a>
