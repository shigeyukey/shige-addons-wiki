Auto adds outline shadow and bold formatting to Anki text and card text for easy to read text even with background colors or background images or wallpaper.

![alt text](https://shigeyukey.github.io/shige-addons-wiki/images/text-shadow/00.png)

## How to use

* After installing the add-on, it will activate automatically.
* Text will be bold, if you don't need it please disable it in Config.
* If text fonts do not support bold text will not be bold.
* Supports light mode and dark mode.
* Outline color automatically changes based on text color.
* If this add-on conflicts with other add-ons, please contact me.
* This add-on works only on PC, it does not affect card data or templates.


## Config

* `is_enable_bold` Toggles whether text is bold or not. value:`true` or `false`.

## üì• How do I install this add-on?

1. This add-on works with Anki for desktop (it does not work on mobile). Download Anki from the [official website](https://apps.ankiweb.net/) if you haven't already.
2. Copy and paste the add-on code ( `1502258980` )  into Anki and you can install it. ( *Menu -> Tools -> Add-ons -> Get Add-ons -> Code \[ add-on code ]* )
3. When I develop bug fixes, create new features, or compatibility for New Anki, I will notify you and you can install it.
4. Add-ons will be broken when the official Anki gets a major update, so if you like this add-on please support my volunteer development by rating, sharing, and donating. Thank you!
