This add-on is a fixed version for Anki25+ of the addon <a href="https://ankiweb.net/shared/info/679615590" target="_blank">"Answer Button Sounds"</a>, originally created by [Kyle Mills](https://github.com/khonkhortisan) and credit goes to them.

### How to use

It is a simple add-on with only 4 sound files, when you answer a sound will play. You can change the sound by changing the file (mp3).
