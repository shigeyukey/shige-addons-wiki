*Gamify your Anki à la Duolingo! Simple visual, audio, and haptic feedback to enrich your studying. (Not affiliated with the official Duolingo.)*

![alt text](https://shigeyukey.github.io/shige-addons-wiki/images/duolingo-style-gamification/00.gif)


This add-on is a Fixed and Customized version for Anki25+ of the addon <a href="https://ankiweb.net/shared/info/255048658" target="_blank">"Duolingo Style Gamification"</a>. The original add-on is no longer maintained and is broken in the latest version of Anki, instead you can use this my version. Originally created by <a href="https://github.com/avanteGarbage" target="_blank">avanteGarbage</a> and credit goes to them.

# How to use

*(these images and descriptions are re-uploaded and partially edited from the original add-on page.)*

Each feature can be disabled individually from the add-on config window.

* Updated review buttons
* Popup feedback
* Chimes for incorrect/correct answers
* Controller vibration!
* You can enable or disable the Add-on from the menu.

## Notes

* The Rumble feature is currently in beta. To support macOS and Linux I've added a V2 mode and it's enabled by default. You can disable this in the options. (V1 supports Windows only.)
* If you change options during the review the changes will take effect after you end the review.

## 📥 How do I install this add-on?

1. This add-on works with Anki for desktop (it does not work on mobile). Download Anki from the [official website](https://apps.ankiweb.net/) if you haven't already.
2. Copy and paste the add-on code ( `1358011657` )  into Anki and you can install it. ( *Menu -> Tools -> Add-ons -> Get Add-ons -> Code \[ add-on code ]* )
3. When I develop bug fixes, create new features, or compatibility for New Anki, I will notify you and you can install it.
4. Add-ons will be broken when the official Anki gets a major update, so if you like this add-on please support my volunteer development by rating, sharing, and donating. Thank you!
