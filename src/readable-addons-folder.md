*This add-on is intended for developers, not general learners, and it is a customized version of "Readable Addons Folder" (fork).*

This Add-on is intended to make it easier for developers to read the Add-on folder name. Add-on folder names are numeric and unreadable to humans. This add-on creates symbolic links or junctions using the actual add-on names.



## How to use

* When Anki is launched, it auto creates folders and symbolic links (MacOS, Linux). On Windows it creates junctions because symbolic links cannot be created on Windows without administrator permissions.
* These are saved in a new folder created within the Anki2 directory. (the same location as the addons21 folder)
* If you delete an add-on its symbolic link will be auto deleted.
* The folder name can be changed in options. Default setting is "Readable Addons by Shige".
* You can manually run the add-on using the action in its menu.
* You can disable startup execution in the Options.

#### Q. How does this add-on differ from the original?

The original add-on is designed for Mac and Linux and does not work on Windows, the forked version supports Windows but it didn’t work properly on my device, so my version resolves these issues. In short this is an add-on I developed for my own use on Windows. For MacOS and Linux users the original add-on should work fine so I recommend using that one.

## 📥 How do I install this add-on?

1. This add-on works with Anki for desktop (it does not work on mobile). Download Anki from the [official website](https://apps.ankiweb.net/) if you haven't already.
2. Copy and paste the add-on code ( `158759867` )  into Anki and you can install it. ( *Menu -> Tools -> Add-ons -> Get Add-ons -> Code \[ add-on code ]* )
3. When I develop bug fixes, create new features, or compatibility for New Anki, I will notify you and you can install it.
4. Add-ons will be broken when the official Anki gets a major update, so if you like this add-on please support my volunteer development by rating, sharing, and donating. Thank you!
