[ üìñ How To Use ]

Show calendar and streaks of New cards learned, like the review heatmap (Prototype). It can be used to visualize learning new cards and make it a habit.

  1. Show number of New cards learned today.

  2. Show longest streak. (New cards only)

  3. Show current streak. (New cards only)

  4. The color intensity changes every 10 cards. The maximum is 100.

  1. Mouse hover tooltip shows date and number of cards.

  2. Heatmap size is the same as the Review heatmap.

  3. Supports night mode.

Heatmap can be squared in config.

  1. Tools -> Addons -> Select New card heatmap -> "border_radius": true or false

[ calculation ]

  1. Add new cards does not count, learning new cards counts.

  2. When the new card interval becomes the next day or later, the number is counted.

  3. If the card does not graduate, it will not be counted. If it graduates the next day, it will be counted then.

[ Noets ]

  1. Color is blue only at this time.

  2. This add-on is relatively simple code, so it does not have advanced features like the Review Heatmap. (This add-on is not a Fork or customized version)

  3. Old Anki is not supported. (like Anki2.1.35 - 2.1.45)
