Confirm deletion

❓️Confirm deletion of the card in Reviewer (Fixed by Shigeඞ)

AnkiWeb Page | Code : 354810378

Show a pop-up before deleting a note in Reviewer to prevent accidental deletion.

This add-on is a fixed and customized version for Anki25+ of the addon "confirm deletion", originally created by ijgnd and credit goes to them.

How to use

  • Auto displays a popup when deleting a note using the reviewer's shortcut key or More button.
  • You can optionally change which button the popup focuses by default (Yes or No). Unchecked defaults to Yes, checked defaults to No.
  • You can optionally change the popup's ask text. Default:"Really delete the current note and all of its cards?"

Note

  • This add-on only works with Reviewer for now, not the browser.
  • This add-on is relatively fragile, so if it doesn't work fine, please contact me.

📥 How do I install this add-on?

  1. This add-on works with Anki for desktop (it does not work on mobile). Download Anki from the official website if you haven't already.
  2. Copy and paste the add-on code ( 354810378 ) into Anki and you can install it. ( Menu -> Tools -> Add-ons -> Get Add-ons -> Code [ add-on code ] )
  3. When I develop bug fixes, create new features, or compatibility for New Anki, I will notify you and you can install it.
  4. Add-ons will be broken when the official Anki gets a major update, so if you like this add-on please support my volunteer development by rating, sharing, and donating. Thank you!
