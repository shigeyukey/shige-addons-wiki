üìñHow to use :

  1. Zoom values for Reviewer, Home, Overview, Editor, Stats can be memorized.
    This effect is valid after restarting Anki.
  2. Auto saving of zoom levels with Ctrl + Scroll Wheel. (Reviewer, Home, Overview, Editor, Stats.)
  3. Separate zoom levels for Question and Answer, it can be disabled in options.
  4. Use shortcut keys to zoom in, zoom out, Reset.(Reviewer, Home, Overview.)
  5. Added option to manually set zoom.(Disable auto save.)

‚öôOptions :


1. Option window
    (Tools -> Zoom for Anki 24)
2. custom options for shortcut keys

Default Shortcut Keys (Home and Reviewer)
  1. Zoom_in : Ctrl+Shift++
  2. Zoom_out : Ctrl+Shift+-
  3. Reset : Ctrl+Shift+^

üö®Known issue:

  1. Incremental reading add-on is incompatible. (Incremental read add-on has its own zoom).
  2. "Ctrl++" or "Ctrl+-" will not work, please set another Shortcut Keys.

Others

  1. I added simple support for old Anki (Anki2.1.35-Anki2.1.44), but some new features are not available.
  2. I added a workaround for a crash problem that was occurring with another zoom addons, so it is relatively safe.
